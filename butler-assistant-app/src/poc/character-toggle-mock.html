UI 設計メモ

  • 非表示ボタン: キャラクターエリア右下の丸ボタン（目に斜線アイコン）。控えめだが見つけやすい位置
  • 折りたたみ時: Live2D エリアが 0 幅/高さにアニメーション縮小（CSS transition 0.35s）。PixiJS の app.stop() で描画を停止し、GPU/バッテリー負荷を解消
  • 天気バッジ: 折りたたみ後はチャットエリア左上にフローティングバーとして残る（天気アイコン + 気温 + 展開ボタン）
  • 展開ボタン: 人型アイコンの丸ボタン。押すとキャラクターエリアを復元し app.start() で描画再開
  • 状態の永続化: appStore の config.ui.characterVisible に保存（Zustand persist）。再起動後も非表示を維持
  • PC: 左カラム（320px）が消え、チャットエリアがフル幅に拡大
  • スマホ: 上部（25vh）が消え、チャットエリアが縦いっぱいに拡大。表示可能なメッセージ量が大幅に増加
  • バッテリー効果: Live2D 非表示時は PixiJS レンダリング完全停止。WebGL コンテキストの GPU 負荷がゼロに
