# 実装計画：執事アプリケーション（Butler Assistant App）

## 概要

本タスクリストは、Live2D技術を用いた感情豊かなキャラクター表現と、LLM（Gemini/Claude）による自然な対話機能を組み合わせた、クロスプラットフォーム対応の執事アプリケーションの実装計画です。

技術スタック：React + TypeScript、Cubism SDK for Web、Gemini/Claude API、Tauri（デスクトップ）、Capacitor（モバイル）、Zustand、Tailwind CSS、Vite

- `*`マークが付いたサブタスクはオプションであり、MVP（最小限の実装）では省略可能です
- 各タスクは具体的な要件番号を参照しており、トレーサビリティを確保しています
- チェックポイントタスクでは、実装の進捗を確認し、問題があれば早期に対処します
- プロパティベーステストは、fast-checkを使用して最低100回の反復実行を行います
- すべてのエラーはログファイルに記録され、ユーザーには適切なメッセージが表示されます
