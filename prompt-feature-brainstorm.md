## アプリ概要

**Butler Assistant** は、Live2D キャラクター × LLM（Gemini / Claude）× 音声合成（Amazon Polly）を組み合わせた、クロスプラットフォーム対応のパーソナルアシスタントアプリです。

画面上に Live2D キャラクター（16歳の明るい女の子）が常駐し、テキストまたは音声でユーザーと会話します。キャラクターは会話内容に応じてモーション（お辞儀、微笑み、頷き等）や表情（喜び、驚き、照れ等）をリアルタイムに変化させ、返答を日本語音声で読み上げます（口パク連動あり）。

## 現在実装済みの機能一覧

### 1. チャット機能
- テキスト入力によるチャット（Enter 送信、Shift+Enter 改行）
- LLM プロバイダー切り替え（Gemini / Claude）
- 会話履歴の保持（最大50メッセージ、LocalStorage 永続化）
- サーバーサイド同期（DynamoDB にメッセージ保存）
- ストリーミング非対応（リクエスト→レスポンス形式）

### 2. Live2D キャラクター
- PixiJS ベースの Live2D Cubism SDK 描画
- 5種類のモーション: idle / bow / smile / think / nod
- 8種類の感情表現: neutral / happy / sad / angry / surprised / thinking / embarrassed / troubled
- LLM がレスポンスごとにモーションと感情を自動選択
- モーション・表情の手動再生パネル
- 表情は3秒後に自動で neutral に戻る
- カスタム Live2D モデルのインポート機能（ドラッグ&ドロップ）

### 3. 音声機能
- **TTS（読み上げ）**: Amazon Polly（Kazuha / neural）による日本語音声合成
  - 返答の自動読み上げ（トグルで ON/OFF）
  - 過去メッセージの個別読み上げボタン
  - リップシンク: 音声の RMS 音量エンベロープを事前計算し、60fps で口の開閉パラメータに反映
- **STT（音声入力）**: Web Speech API（ブラウザネイティブ）
  - マイクボタンで音声認識開始/停止
  - 認識中のリアルタイム中間テキスト表示
  - 音声自動送信モード: 認識結果確定後 3.5秒で自動送信

### 4. キャラクター設定
- システムプロンプトで性格を定義（16歳、タメ口、明るい性格）
- 絵文字不使用（感情は Live2D の表情で表現）

### 5. ユーザープロフィール
- ニックネーム登録 → キャラがその名前で呼んでくれる
- 敬称設定（「くん」「さん」「ちゃん」等）
- 性別設定 → 会話の個性化に反映

### 6. UI 設定
- ダークモード / ライトモード切り替え
- フォントサイズ調整
- TTS ON/OFF

### 7. 認証・データ同期
- Amazon Cognito 認証（メール + SRP）
- サインアップ / ログイン / パスワードリセット
- ログイン時にサーバーと設定・メッセージを自動同期
- Cognito 未設定時はゲストモード（ローカルのみ）

### 8. エラーハンドリング
- エラー種別ごとのキャラクター反応（モーション + キャラクターの言葉でのエラー説明）
- トースト通知（5秒自動非表示）
- 指数バックオフリトライ（LLM API）

## 技術スタック

- **フロントエンド**: React 19 + TypeScript 5.9 + Vite 7 + Tailwind CSS 4 + Zustand 5
- **Live2D**: pixi.js 7 + pixi-live2d-display
- **認証**: AWS Amplify 6 + Cognito
- **バックエンド**: AWS CDK（API Gateway + Lambda Node.js 22 + DynamoDB + Polly）
- **テスト**: Vitest + jsdom + fast-check（343テスト / 20ファイル）

## 現在のアーキテクチャ

```
[ユーザー] ←テキスト/音声→ [ChatUI]
                              ↓
                        [chatController]
                         ↓         ↓          ↓
                    [llmClient]  [ttsService]  [motionController]
                      ↓              ↓               ↓
                  Gemini/Claude   Polly API      Live2D SDK
                      ↓              ↓               ↓
                  構造化JSON      音声(MP3)     モーション/表情
                  {text,motion,   リップシンク    描画更新
                   emotion}       口パク連動
```

## 提案してほしいこと

以下の観点から、**実現可能性が高く、ユーザー体験を大きく向上させる機能**を提案してください：

1. **日常的に使いたくなる実用機能** — 毎日開く理由になるもの
2. **キャラクターとの関係性を深める機能** — 愛着・親密感を育むもの
3. **技術的に面白い挑戦** — Live2D + LLM + 音声の組み合わせならではのもの
4. **短期間で実装可能な改善** — 1〜3日で効果が出るクイックウィン

各提案には以下を含めてください：
- 機能の概要と利用シーン
- ユーザーにとっての価値（なぜ嬉しいか）
- 技術的な実現方針の概要
- 実装の難易度（小/中/大）
- 既存のアーキテクチャとの相性
